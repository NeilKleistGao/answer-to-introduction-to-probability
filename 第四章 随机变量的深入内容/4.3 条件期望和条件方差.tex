    \section*{22.} 
        设第i次赌博的收益比率为常量$R_i$，资产值为随机变量$X_i$，我们可以得到：
        \begin{itemize}
            \item $E[X_1] = (pR_1 + 2(1 - p)^2)x$
            \item $E[X_2] = E[E[X_2|X_1]] = (pR_2 + 2(1 - p)^2)X_1$
            \item $E[X_3] = E[E[X_3|X_2]] = (pR_3 + 2(1 - p)^2)X_2$
            \item \dots
        \end{itemize}

        综上：$E[X_n] = x\Pi_{i = 1}^n(pR_i + 2(1 - p)^2)$
